Rapi::Fs

Rapi::Fs is a Plack-compatible file browser application written using RapidApp. It generates a nice-looking dynamic web interface to browse and view arbitrary files and directories for the configured "mounts" using a standard, split-panel tree layout. More info and a live demo is available on the Rapi::Fs project home page at rapi.io/fs.

Live Demo Site: rapi.io/fs
