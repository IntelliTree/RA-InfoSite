RapidDbic "Chinook" Demo

This video demo series provides a tutorial of RapidApp database features via the "RapidDbic" plugin, using the 3rd-party "Chinook" sample database (chinookdatabase.codeplex.com) for the backend.

In this demo, a fully functioning database-driven app is created from scratch with every change and command recorded to the companion Github repository along the way. Commands and text-comments are recorded to the file cmd_history.sh within the repo.

Companion Git Repo (code & commands):

github.com/IntelliTree/RA-ChinookDemo

Step-by-step Command History:

cmd_history.sh
